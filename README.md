# Customer Retention Cohort Analysis

This project focuses on customer retention analysis using SQL and Google Sheets through cohort analysis techniques.

The project includes:
- Cleaning and standardizing inconsistent date formats using SQL
- Building reusable SQL queries with CTEs and JOINs
- Calculating monthly customer cohorts and month offsets
- Preparing aggregated retention datasets
- Creating interactive cohort tables in Google Sheets
- Calculating Retention Rate for promotional and organic users
- Applying conditional formatting and interactive slicers
- Comparing customer behavior based on acquisition channels
